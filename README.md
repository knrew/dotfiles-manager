# dotfiles_installer

## install
```sh
cargo install --path .
```

## run
### help
```sh
dotfiles_installer --help
```

### run
```sh
dotfiles_installer --dotfiles /path/to/dotfiles_dir --install /path/to/install_dir --backup /path/to/backup_dir
```

### exsample
```sh
dotfiles_installer --dotfiles ~/dotfiles --install ~/ --backup ~/.backup_dotfiles
```
